TourneyPro screenshots for the portfolio

Copy your TourneyPro app screenshots into this folder with these names so they appear on the Featured Systems card and in the project modal:

  hero.png           - Homepage / "Kick off your next tournament!" (main card image)
  group-standings.png - Group A table (Pos, Team, P, W, D, L, GF, GA, GD, PTS)
  event-page.png     - Fantasy Premier 6s event (date, location, R400, Add Team)
  individual-stats.png - Individual Stats tabs (Top Scorers, Assists, Clean Sheets, G+A/Game)
  team-profile.png   - Team overview (e.g. Volta FC or Island Boys, Players, Record, Points)
  leaderboard.png    - G+A/Game leaderboard (player rankings)

If hero.png is missing, the portfolio will show a fallback image until you add it.
